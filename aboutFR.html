Textcrypter est une petite application web qui vous permet de créer des messages codés et de les décoder. C'est très simple. Entrez votre texte dans la zone de texte "entrée", choisissez un nombre et cliquez sur crypter !

Envoyez un message codé a un ami, celui-ci pourra le décoder sur ce site en collant votre text encodé dans la zone de texte "entrée". Pour pouvoir décoder correctement le message votre ami devra connaître le nombre de rails que vous aviez choisi précédemment.

Essayez aussi de changer le nombre de rails quand vous encodez votre message pour obtenir différent résultats. Cependant, le nombre de rails doit être plus petit que le nombre de caractère contenu dans votre texte.

Amusez vous bien!
