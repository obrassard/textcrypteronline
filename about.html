Textcrypter Is a little web app that allows you to create coded message and decrypt them. It is super simple. Enter your text in the "input" text box, chose a number and click on encrypt!

Send a coded message to a friend, this one will be able to decrypt it on this website by pasting your encoded text in the "input" text field. In order to decrypt correctly your message your friend must know the number of rails you chosen to encrypt it.

Try changing the number of rails when you encrypt a message to get different output. However the number of rails must be smaller than the number of characters that your text contains.

Enjoy !
